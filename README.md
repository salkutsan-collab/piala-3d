# Конструктор-полигон: пиала и блюдце (3D-демо)

Интерактивный просмотрщик: пиала и блюдце в 3D с наложением декора-деколи, проверкой
макета и предкомпенсацией. Сделано для концепции конкурса "Дизайн молодых".

## Как открыть

- Локально: открыть `index.html` двойным кликом (нужен интернет - библиотека three.js грузится с CDN).
- В сети: включить GitHub Pages (см. ниже) и открыть выданную ссылку.

## Что внутри

- Реальные формы: пиала D110 H55 (полусфера), блюдце D150 с углублением и бортом.
- Декор наложен текстурой, внутренность белая, у стенок есть толщина.
- Панель замечаний (справа): файл и оборудование, искажения, нахлест и шов.
- Кнопка "Предкомпенсация" - заранее искаженный рисунок, чтобы лег ровно.

## Управление

- Тянуть мышью - вращать; колесо - зум; правая кнопка - сдвиг.
- Кнопки: Пиала / Блюдце / Вместе, и переключатель Предкомпенсация.

## Важно про права

В демо встроен декор-макет ("Татарстан"). Если репозиторий публичный, этот рисунок
станет общедоступным. Для публичной демонстрации лучше заменить его на нейтральный
образец или сделать репозиторий приватным.
